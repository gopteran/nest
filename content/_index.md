## What is Gopteran?

Gopteran is a comprehensive infrastructure management ecosystem designed for **flight and precision**. Our suite of tools provides everything you need to deploy, manage, and scale your applications with confidence.

### Core Components

- **[Aerie](/docs/aerie)** - Fullstack control panel combining frontend and backend management
- **[Carina](/docs/carina)** - Backend core providing authentication and API orchestration
- **[Ventus](/docs/ventus)** - Modern SvelteKit frontend framework for infrastructure management
- **[Remora](/docs/remora)** - Powerful CLI agent and daemon for server-side operations
- **[Avis](/docs/avis)** - Discord bot for community communication and automation
