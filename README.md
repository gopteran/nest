## SEO Configuration

This site includes a comprehensive SEO configuration system optimized for technical documentation:

### Key Features
- **Advanced Meta Tags**: Complete Open Graph, Twitter Cards, and structured data
- **Multi-Platform Verification**: Google, Bing, Yandex, and Baidu search console integration
- **Analytics Integration**: Google Analytics 4, Google Tag Manager, and social media pixels
- **Performance Optimization**: Core Web Vitals monitoring and resource preloading
- **Internationalization**: Multi-language support with hreflang tags
- **Rich Snippets**: FAQ, How-To, and Article schema markup
- **Social Media Optimization**: Optimized sharing across all major platforms

### Configuration Files
- `config/_default/seo.toml` - Comprehensive SEO settings
- `config/_default/params.toml` - General site parameters
- `layouts/_default/robots.txt` - Dynamic robots.txt generation
- `layouts/_default/sitemap.xml` - Enhanced XML sitemap with rich snippets

### Quick Setup

1. Update domain-specific settings in `config/_default/seo.toml`
2. Add your actual tracking IDs and verification codes
3. Configure social media handles and app IDs
4. Enable desired features (video SEO, e-commerce, local SEO)

For detailed configuration instructions, see `SEO_GUIDE.md`.

## Purpose

- Serve as the authoritative source for all Gopteran documentation
- Support users and contributors with clear guides, tutorials, and references
- Act as the content base for the official Gopteran documentation website
